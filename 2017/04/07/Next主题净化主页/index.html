Next主题净化主页

发表于 2017-04-07 | 分类于 blog

#前言

Next默认主题主页导航内容有些啰嗦，而且时常会因为<!-- more -->出BUG导致排版很难看。所以决定把主页净化一下。
主页的作用很简单，就是只要显示标题，时间，类型就可以了。来看看净化前的效果

#优化

打开themes/next/layout/_macro/post.swig 注释掉选中内容

打开themes/next/source/css/_schemes/Mist/_posts-expanded.styl，修改样式

我用的是Mist样式。其他样式可以找对应样式目录

#优化后

明显导航干净很多，突出重点！

注：
有其他需要修改的内容，可以在post.swig内搜索is_index参数，这个参数表示是导航页
