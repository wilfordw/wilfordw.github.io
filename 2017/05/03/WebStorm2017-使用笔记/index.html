#基础

  • Find Actions ⇧⌘A
  • Insert Live Template ⌘J
  • Refactor this ⌃T"
  • Goto Next Splitter ⌥TAB
  • Goto Prevous Splitter ⌥⇧TAB
  • Reformat Code ⌃⌘L

  • Indent Line or Selection TAB

  • Move Statement Down ⌘⇧DOWN
